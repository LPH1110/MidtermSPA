-------- HƯỚNG DẪN START PHẦN MỀM BETTERTRIP --------
1. Cài Nodejs vào máy tính của bạn

2. Tại thư mục source của project, vào cmd tại thư mục gốc server 
-> chạy lệnh "npm i" để install node_modules và các packages cần thiết khác 
-> sau khi cài xong, chạy lệnh "npm start"
-> sau khi có thông báo kết nối đến Database thành công thì thực hiện bước tiếp theo

3. Tại thư mục source của project, vào cmd tại thư mục gốc client 
-> chạy lệnh "npm i" để install node_modules và các packages cần thiết khác 
-> sau khi cài xong, chạy lệnh "npm start"
-> đợi webpack compile xong sẽ tự động mở ra tab localhost

4. Sử dụng phần mềm

***Note: Trong quá trình start phần mềm có xảy ra lỗi vui lòng feedback lại cho admin dự án hoặc admin source
